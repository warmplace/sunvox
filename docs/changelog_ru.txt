v2.1.2 BETA1 (3 сентября 2024):
* новый модуль Smooth - пытается повторить волну входящего сигнала, сглаживая резкие перепады в соответствии с параметрами ПОДЪЕМ и СПАД;
* новые типы модуляции в модулях FMX и Analog Generator (Осц2 режим):
  * min abs: if( abs(in2) < abs(in1) ) out = in2; else out = in1;
  * max abs: if( abs(in2) > abs(in1) ) out = in2; else out = in1;
* Modulator: добавлены дополнительные типы модуляции, как в FMX и Analog Generator;
* LFO: улучшенное качество волновых форм "пила", "треугольная" и "случайная плавная" (26 бит вместо 15);
* Filter Pro: максимум контроллера "Крутизна" увеличен до 96дБ;
* Filter Pro: начиная с версии 2.1.2 контроллер "Крутизна" влияет в том числе на тип фильтра "фазовый"; см. simple_examples/freq dispersion.sunvox;
* Reverb: добавлен улучшенный тип фазового фильтра, который почти не влияет на амплитуды частот исходного сигнала; выбирается контроллером 8;
* Delay: новый контроллер "Отрицательная обратная связь" меняет знак обратной связи;
* Delay: новый контроллер "Фазовый фильтр" переключает Delay в режим фазового фильтра, где порядок фильтра - это длина задержки в отсчетах;
* Glide: новые контроллеры "Октава" и "Множитель частоты";
* MultiCtl: добавлена кнопка "Установить" для точной установки значений кривой;
* MultiCtl: добавлена кнопка "+" (напротив номера контроллера) - она создает дополнительный выходной слот и позволяет управлять несколькими контроллерами внутри одного модуля; см. simple_examples/multictl.sunvox;
* Pitch Detector: новый алгоритм - "пик спектра"; очень точно определяет частоту гармоники с максимальной амплитудой; подходит для сигналов с ярко выраженной основной гармоникой;
* Pitch Detector: дополнительная информация под контроллерами: частота входящего сигнала; диапазон частот; частота ноты на выходе;
* Pitch Detector: максимальные детектируемые частоты для алгоритмов 1 и 2 - это 2000 Гц и 1000 Гц; но данные значения удвоятся, если выполнятся следующие условия: Базовая версия >= 2.1.2; Частота НЧ фильтра == 0; Алг1-2 Частота дискретизации >= 44100;
* Sampler: новый контроллер "Запись" (стоп/пауза/старт записи); см. simple_examples/sampler rec automation.sunvox;
* редактор сэмпла: добавлена функция "Установить начало" (позиция, с которой сэмпл будет начинать играть);
* редактор сэмпла: добавлены функции "Конвертировать в 8/16/32бита" (меню редактора -> Прочее);
* редактор сэмпла: настройки отображения (в меню) позволяют задавать, в каких единицах (отчеты, секунды, тики и т.д.) показывать положение курсора и размер выделения;
* редактор сэмпла: добавлена кнопка "Играть" (начиная с позиции курсора, без огибающих);
* редактор сэмпла: улучшенное отображение сэмпла (показываются все пики; самый точный результат будет в версиях arm64 и x86_64+SSSE3 (OpenGL));
* эффект 34 (в колонке EE): добавлена опция 8 - включает 7-битный вывод для всех MIDI-контроллеров, даже если контроллер 14-битный (0-31);
* новые сочетания клавиш (по умолчанию не назначены):
  * предыдущая планировка;
  * следующая планировка;

  * планировка: таймлиния;

  * планировка: модули + контроллеры;
  * планировка: контроллеры;
  * планировка: модули;
* новые сочетания клавиш (по умолчанию не назначены) для редактора модулей:

* новые сочетания клавиш (по умолчанию не назначены) для таймлинии:
  * следующий паттерн по горизонтали;
  * предыдущий паттерн по горизонтали;
  * следующий паттерн по вертикали;
  * предыдущий паттерн по вертикали;
* новые сочетания клавиш (по умолчанию не назначены) для кисти:
  * проиграть текущую ноту (в редакторе паттерна), скопировать в кисть и перейти на следующую строку;
  * проиграть текущую строку (в редакторе паттерна), скопировать в кисть и перейти на следующую строку;
* новые опции в sunvox_config.ini: no_clog - выключить вывод лога (отладочных сообщений) в консоль; no_flog - выключить вывод лога в файл; см. также соотв. опции командной строки -ncl и -nfl;
* Windows: новый звуковой драйвер DirectSound2 позволяет уменьшить размер буфера до 512 - это может быть полезным, если вы не можете использовать ASIO по какой-то причине;
* 64-битные версии (не OpenGL) теперь могут запускаться на процессорах без поддержки SSSE3;
* новые модули и примеры (см. changelog_files.txt);
* обновленная версия библиотеки SunVox для разработчиков: http://warmplace.ru/soft/sunvox/sunvox_dll.zip;
